What is MERN?

Just a quick overview before we dive in, MERN stands for:

M: MongoDB, a noSQL database which stores data persistently in the form of collections and documents, MongoDB will be hosted on MongoDB Atlas

E: Express.js, is a web application framework for Node.js that makes app development in Node easier and faster. Node and Express will work in unison to form the middle-tier of the web server in the stack, this will be hosted on Heroku

R: React.js, React is a JavaScript frontend library developed by FaceBook to build user interfaces, this is where we'll call our API and render the content on the clients browser dynamically, will be hosted on Netlify

N: Node.js, Node is one of the most popular server-side frameworks that allows JS to execute inside a web server, we'll also be hosting on this on Heroku

It's important to note these stacks like MERN, MEA(Angular.js)N and MEV(Vue.js)N are used to build SPA's or single page applications, which is a really cool idea where content is rendered in components dynamically vs reloading a webpage entirely in a traditional website or web app
